# 3PL-Delivery-Management-System---case-Study-inspired-by-a-Major-Convenience-Store-Chain-Circle-K-
A structured systems analysis and design (SAD) case study mapping data flows (DFD) and process specifications for a 3PL last-mile delivery system, inspired by Circle K.

Circle K Business Model & Management Information Systems (MIS) Analysis
An in-depth academic research and systems analysis project focused on the retail operations, supply chain logistics, and Point-of-Sale (POS) systems of **Circle K Vietnam**. This project models information workflows to optimize retail efficiency and data management.

 # Project Objectives
* **Business Process Modeling:** Analyze and map out core retail operations, including sales management and inventory control.
* **Data Flow Architecture:** Design Multi-level Data Flow Diagrams (DFDs) to visualize structural data movements.
* **Data Standardization:** Construct a comprehensive Data Dictionary to define core entities and attributes for retail database management.
